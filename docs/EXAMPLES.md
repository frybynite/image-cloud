# Examples

Live examples are published alongside the docs. Source files are in the `examples/` directory of the repository.

## Loader Examples

| Example | Description |
|---------|-------------|
| [Static URLs Shorthand](https://frybynite.github.io/image-cloud/examples/static-urls-shorthand-example.html) | Simplest setup: pass a direct array of image URLs |
| [Static Image Cloud](https://frybynite.github.io/image-cloud/examples/static-loader-example.html) | Load images from static URLs using the static loader |
| [Static JSON Source](https://frybynite.github.io/image-cloud/examples/static-json-source-example.html) | Fetch image URLs from a JSON endpoint |
| [Google Drive Cloud](https://frybynite.github.io/image-cloud/examples/google-drive-loader-example.html) | Load images from a public Google Drive folder |
